# 🚖 Uber Analytics Dashboard

## 📌 Project Description

The Uber Analytics Dashboard is an interactive Power BI project developed to analyze ride booking data and uncover meaningful business insights. The dashboard provides a comprehensive view of booking performance, revenue generation, vehicle-wise analysis, customer satisfaction, and operational trends through dynamic visualizations and KPI monitoring.

This dashboard enables users to explore booking patterns, evaluate vehicle performance, monitor revenue trends, and understand customer and driver ratings. Interactive filters and navigation features allow users to drill down into specific vehicle categories and gain deeper insights into ride operations.

---

## 🎯 Objectives

- Analyze booking performance and ride trends.
- Monitor completed and cancelled bookings.
- Evaluate revenue generated across different vehicle types.
- Track total and average ride distances.
- Measure customer and driver satisfaction through ratings.
- Provide interactive visualizations for data-driven decision making.
